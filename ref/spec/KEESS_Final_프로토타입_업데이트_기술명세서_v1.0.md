## 2. 대시 기호('—') 전역 제거

### AS-IS
- title: `KEESS — KG에듀원 기업교육 | 진단으로 설계하고, 효과로 증명합니다.`
- meta description: `진단으로 설계하고, 효과로 증명합니다. KG에듀원 HRD사업본부의 기업·기관 교육 도입 채널 — AX·AI 전환·리더십·HRD 통합·콘텐츠 솔루션.`
- 이 외 페이지 본문 카피에도 '—' 사용 가능성 있음

### TO-BE
- 전체 코드베이스에서 em dash(—, U+2014), en dash(–, U+2013), 가로선 문자(ㅡ, ―, U+2015)를 검색하여 문맥에 맞게 치환. 기계적 일괄 치환 금지, 문장별 자연스러운 연결로 수정
- title 변경안: `KEESS | KG에듀원 기업교육 · 진단으로 설계하고, 효과로 증명합니다`
- meta description 변경안: `진단으로 설계하고, 효과로 증명합니다. KG에듀원 HRD사업본부의 기업·기관 교육 도입 채널, AX·AI 전환·리더십·HRD 통합·콘텐츠 솔루션.`
- 치환 규칙 가이드
  - `A — B(부연 설명)` 형태: `A. B` 또는 `A: B` 또는 조사로 자연 연결
  - 인용 카피 뒤 `” — 설명` 형태: 대시 제거 후 공백으로 연결
  - 날짜·범위 표기는 대시 대신 `~` 사용
- OG 태그(og:title, og:description) 등 소셜 메타에 동일 문구가 있으면 함께 수정

### 수용 기준
- [ ] `grep -rn "—\|–\|―"` 결과 0건 (소스 전체 기준)
- [ ] 브라우저 탭 타이틀·검색 미리보기(메타)에서 대시 미노출
- [ ] 치환 후 문장 어색함 없음(빌드 후 전 페이지 육안 검수)

## 3. 홈 References 섹션: 고객사 CI 롤링 개편

### AS-IS
- "이미 이렇게 함께하고 있습니다." 아래 6개 분류(그룹/산학/협회·조합/보험/증권·카드/제조·생산) 정적 그리드, 분류당 플레이스홀더 SVG 5개, "(노출 협의 기준)" 주석

### TO-BE
- 정적 그리드를 **무한 롤링(마퀴) 배너**로 변경
- 레이아웃: 로고를 2~3개 행(row)으로 나누어 가로 무한 롤링. 행별 진행 방향 교차(1행 좌→우, 2행 우→좌) 권장
- 동작 명세
  - 트랙 복제 방식(로고 리스트 2회 렌더)으로 이음새 없는 무한 루프
  - 속도: 행당 40~60초/1루프 수준의 저속, hover 시 일시정지
  - `prefers-reduced-motion: reduce` 환경에서는 롤링 중지 후 정적 그리드로 대체(접근성)
  - 로고는 기본 grayscale 처리, hover 시 원색 전환(선택 사항, 톤 통일 목적)
- 콘텐츠: 고객사 CI는 제안서 래퍼런스 기준(그룹: 동원·롯데·CJ·아주·LS·웅진·KG / 은행: KB금융그룹·신한은행·우리은행·NH농협은행·KDB산업은행·BNK부산은행·DGB대구은행·전북은행 / 협회·조합: 농협중앙회·신협중앙회·수협중앙회·저축은행중앙회·새대한공인중개사협회·대한치과의사협회·한국공인중개사협회·한국교직원공제회 / 보험: 삼성생명·삼성화재·신한라이프·한화생명·동양생명·하나손해보험·메리츠화재·KB생명보험 / 증권·카드: KB증권·한국투자증권·신한투자증권·신한카드·교보증권·IBK투자증권·SK증권·롯데카드 / 제조·생산: 동국제강·금호석유화학·POSCO·벽산·Haatz·도레이첨단소재·KYOCERA·KGM)
- CI 이미지 파일 수급 전까지는 사명 텍스트 로고 칩(placeholder)으로 구현하고, `/public/logos/` 경로 규칙과 데이터 배열(배열에 항목 추가 시 자동 반영 구조)만 갖춰둘 것

## 4. 푸터 ISMS 인증서 확인 기능

### AS-IS (참고: 기존 keess.co.kr 푸터 분석 결과)

- Final 프로토타입: ISMS 배지가 이미지로만 노출되며 클릭 동작 없음

### TO-BE
- 푸터 ISMS 배지를 클릭 가능한 버튼으로 변경(cursor: pointer, 키보드 포커스 가능)
- 클릭 시 '정보보호 관리체계(ISMS) 인증서'를 모달(라이트박스)로 표시
- 모달 명세
  - 콘텐츠: 인증서 이미지 1장 + 상단 타이틀("정보보호 관리체계(ISMS) 인증서") + 닫기 버튼
  - 닫기: X 버튼, 오버레이 클릭, ESC 키 3가지 모두 지원
  - 접근성: `role="dialog"`, `aria-modal="true"`, 열릴 때 포커스 이동·닫힐 때 원위치 복귀, 배경 스크롤 잠금
  - 이미지: 세로형 인증서 비율 유지, 뷰포트 초과 시 내부 스크롤
- 자산: 인증서 이미지는 기존 사이트의 `certificate_isms.jpg`와 동일 원본을 `/public/certificates/isms.jpg`로 프로젝트에 포함(외부 URL 핫링크 금지). 원본 파일은 별도 전달 예정이며, 수급 전까지 동일 비율의 placeholder 사용
- 본 기능은 공통 푸터 컴포넌트에 구현하여 홈·P1~P4 전 페이지에서 동일 동작
